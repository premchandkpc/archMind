## What this chunk does

Assembles the LangGraph StateGraph with nodes and edges.
Implements conditional routing: planner decides which nodes to run.
Reviewer decides whether to loop back or proceed to report.

## Files to create

### civilmind/workflow/graph.py

**Purpose:** build_graph(), route_after_planner(), route_after_review()

## Graph Structure

```
START → planner → router
                    |
        +-----------+-----------+
        |           |           |
    retriever  estimator  compliance
        |           |           |
        +-----------+-----------+
                    |
                  reviewer
                    |
            +-------+-------+
            |               |
         reporter      planner (loop back if issues)
            |
           END
```

## Routing Functions

### route_after_planner

```python
async def route_after_planner(state: ProjectState) -> str:
    """Route based on planner decision."""
    next_nodes = state.get("next_nodes", ["retriever"])
    node_map = {
        "retrieval": "retriever",
        "drawing_analysis": "drawing_analyzer",
        "compliance_check": "compliance",
        "estimation": "estimator",
        "scheduling": "scheduler",
        "risk_analysis": "risk_analyzer",
    }
    mapped = [node_map.get(n, n) for n in next_nodes]
    return mapped[0] if mapped else "reviewer"
```

**Why node_map:**
- Planner returns human-readable names ("retrieval")
- LangGraph needs exact node names ("retriever")
- Maps between the two

### route_after_review

```python
async def route_after_review(state: ProjectState) -> str:
    """Route based on review feedback."""
    review = state.get("review_feedback", {})
    iteration = state.get("iteration", 0)
    
    if review.get("is_valid", False):
        return "reporter"
    if iteration < MAX_ITERATIONS:
        return "planner"
    return "reporter"
```

**Why loop back to planner:**
- Reviewer found issues (missing docs, low confidence)
- Planner can adjust the plan (try different approach)
- Max 3 iterations to prevent infinite loops

## Graph Builder

```python
def build_graph(checkpointer=None) -> StateGraph:
    graph = StateGraph(ProjectState)
    
    # Nodes
    graph.add_node("planner", planner_node)
    graph.add_node("retriever", retriever_node)
    graph.add_node("drawing_analyzer", drawing_analyzer_node)
    graph.add_node("compliance", compliance_node)
    graph.add_node("estimator", estimator_node)
    graph.add_node("scheduler", scheduler_node)
    graph.add_node("risk_analyzer", risk_analyzer_node)
    graph.add_node("reviewer", reviewer_node)
    graph.add_node("reporter", reporter_node)
    graph.add_node("human_approval", human_approval_node)
    
    # Entry
    graph.set_entry_point("planner")
    
    # Edges
    graph.add_conditional_edges("planner", route_after_planner, {
        "retriever": "retriever",
        "drawing_analyzer": "drawing_analyzer",
        "compliance": "compliance",
        "estimator": "estimator",
        "scheduler": "scheduler",
        "risk_analyzer": "risk_analyzer",
        "reviewer": "reviewer",
    })
    
    graph.add_edge("retriever", "reviewer")
    graph.add_edge("drawing_analyzer", "reviewer")
    graph.add_edge("compliance", "reviewer")
    graph.add_edge("estimator", "reviewer")
    graph.add_edge("scheduler", "reviewer")
    graph.add_edge("risk_analyzer", "reviewer")
    
    graph.add_conditional_edges("reviewer", route_after_review, {
        "planner": "planner",
        "reporter": "reporter",
    })
    
    graph.add_edge("reporter", END)
    graph.add_edge("human_approval", END)
    
    return graph.compile(checkpointer=checkpointer or MemorySaver())
```

## Usage

```python
from civilmind.workflow.graph import build_graph
from civilmind.workflow.state import create_initial_state

graph = build_graph()

# Synchronous
result = graph.invoke(create_initial_state(
    project_id="123",
    question="Estimate concrete for this building",
))

# Asynchronous
result = await graph.ainvoke(
    create_initial_state("123", "Estimate concrete"),
    config={"configurable": {"thread_id": "run-1"}},
)

# Stream
async for event in graph.astream(initial_state):
    print(event)
```

## Checkpointing

```python
from langgraph.checkpoint.postgres import PostgresSaver

# For production, persist state to PostgreSQL
checkpointer = PostgresSaver(connection_string=settings.DATABASE_URL)
graph = build_graph(checkpointer=checkpointer)

# State is automatically saved after each node
# Can resume from any checkpoint
```
