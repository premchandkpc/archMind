## What this chunk does

Measures answer quality: is the LLM hallucinating? Is the answer faithful to retrieved context?

## Metrics

| Metric | Definition | How |
|--------|-----------|-----|
| Faithfulness | % of answer supported by context | LLM-as-judge: "Is each claim in the answer supported by the context?" |
| Answer Relevance | Does the answer address the question? | LLM-as-judge: "Does this answer the original question?" |
| Hallucination Rate | % of claims not in context | Extract claims, verify against context |
| Citation Accuracy | % of cited sources that support the claim | Check each citation matches the claim |

## LLM-as-Judge Prompt

```
You are a fact-checking assistant. Given a QUESTION, ANSWER, and CONTEXT,
determine what percentage of the ANSWER's claims are supported by the CONTEXT.

For each claim in the answer:
- SUPPORTED: The claim is directly supported by the context
- NOT SUPPORTED: The claim cannot be verified from the context
- CONTRADICTED: The claim contradicts the context

Return JSON:
{
  "score": 0.85,
  "total_claims": 10,
  "supported_claims": 8,
  "unsupported_claims": ["claim1", "claim2"]
}
```
