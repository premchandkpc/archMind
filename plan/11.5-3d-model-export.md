# 11.5 — 3D Model Export: SketchUp + Blender + OBJ

**Status:** Planned

---

## What this chunk does

Exports the converted geometry into formats used by popular 3D modeling tools:
- **SKP** — SketchUp native format (via `sketchup` API or Collada→SKP conversion)
- **BLEND** — Blender project file (via `bpy` / command-line Blender)
- **OBJ** — Universal 3D mesh format (works in Maya, 3ds Max, Cinema4D, Unity, Unreal)
- **GLTF/GLB** — Web-friendly 3D format (for browser previews)
- **STL** — 3D printing format (for physical scale models)
- **COLLADA (DAE)** — Intermediate exchange format (works with most tools)

## Why these formats

| Tool/Format | Use case | Users |
|---|---|---|
| **SketchUp** | Quick 3D model from 2D plans | Architects, structural engineers |
| **Blender** | Free, full-featured 3D modeling + rendering | Visualization teams |
| **OBJ** | Universal mesh, works everywhere | Downstream pipelines |
| **GLTF/GLB** | Web preview in browser | Client presentations |
| **STL** | 3D printing of scale models | Physical mockups |

## Pipeline

```
CadElement[] (from Chunk 11.2)

  1. Extrude 2D geometry → 3D mesh (triangulated)
  2. Export to universal intermediate (OBJ)
  3. Convert OBJ → target format:
       ├── GLTF/GLB   (via trimesh or pygltflib)
       ├── STL        (via trimesh or numpy-stl)
       ├── COLLADA     (via pycollada or trimesh)
       └── SKP         (via SketchUp Ruby API or Collada→SKP)
  4. Optionally generate Blender Python script (.py)
     that recreates the model (for BLEND export)
```

## Files to create

### civilmind/cad/extruder.py

**Purpose:** Convert 2D CadElement footprints into 3D meshes.

```python
from __future__ import annotations

import numpy as np
from dataclasses import dataclass
from typing import Any

from civilmind.cad.models import CadElement, Point

@dataclass
class Mesh3D:
    """Triangulated 3D mesh."""
    vertices: list[tuple[float, float, float]]
    faces: list[tuple[int, int, int]]
    normals: list[tuple[float, float, float]] | None = None
    uv_coords: list[tuple[float, float]] | None = None

class MeshExtruder:
    """Convert CadElements to 3D meshes by extrusion."""

    # Default heights (mm)
    DEFAULT_WALL_HEIGHT = 3000
    DEFAULT_COLUMN_HEIGHT = 3000
    DEFAULT_BEAM_HEIGHT = 450
    DEFAULT_SLAB_THICKNESS = 200
    DEFAULT_DOOR_HEIGHT = 2100
    DEFAULT_WINDOW_HEIGHT = 1200
    DEFAULT_WINDOW_SILL = 900

    def extrude(self, element: CadElement) -> Mesh3D:
        """Extrude a 2D CadElement into a 3D mesh."""

    def extrude_all(self, elements: list[CadElement]) -> list[Mesh3D]:
        """Extrude all elements."""
        return [self.extrude(el) for el in elements]

    def _extrude_polygon(self, polygon: list[Point], height: float) -> Mesh3D:
        """Extrude a 2D polygon by height. Returns triangulated mesh.

        Creates:
        - Bottom face (reverse normal)
        - Top face
        - Side faces (one quad per edge, split into 2 triangles)
        """

    def _extrude_line_as_beam(self, line_start: Point, line_end: Point,
                               width: float, height: float) -> Mesh3D:
        """Extrude a line into a rectangular beam."""
```

Options:
1. **COLLADA (.dae)** — SketchUp imports natively via File → Import. Works today.
2. **SketchUp Ruby script** — `scripts/skp_import.py` generates a `.rb` file:
   ```ruby
   # Generated script — run in SketchUp Ruby Console
   model = Sketchup.active_model
   entities = model.entities
   # Vertex data
   pts = [...]
   face = entities.add_face(pts)
   face.pushpull(3000)  # extrude to wall height
   ```
3. **Cloud conversion** — SketchUp API supports file upload via `https://developer.sketchup.com/`.

### civilmind/cad/blend_exporter.py

**Purpose:** Generate Blender-compatible output.

```python
class BlendExporter:
    """Export to Blender via Python script or COLLADA."""

    def export(self, meshes: list[Mesh3D], output_path: str) -> str:
        """
        Strategy:
        1. Export OBJ (universal)
        2. Optionally generate Blender Python script (.py):
           - Creates meshes, sets materials, applies transforms
           - User runs: blender --python generated_script.py
        3. If bpy available (headless Blender), render directly to .blend:
           blender -b -P scripts/render_blend.py
        """
```

**Blender Python script** (`scripts/export_to_blender.py`):
```python
# Generated script — run with: blender --python this_script.py
import bpy

# Clear scene
bpy.ops.object.select_all(action='SELECT')
bpy.ops.object.delete(use_global=False)

# For each element, create mesh
for el in elements:
    mesh = bpy.data.meshes.new(el.name)
    obj = bpy.data.objects.new(el.name, mesh)
    bpy.context.collection.objects.link(obj)

    # Set vertices and faces
    mesh.from_pydata(el.vertices, [], el.faces)
    mesh.update()

    # Assign material
    mat = bpy.data.materials.new(el.material_name)
    obj.data.materials.append(mat)

# Save
bpy.ops.wm.save_as_mainfile(filepath=output_path)
```

### civilmind/cad/format_detector.py

**Purpose:** Auto-detect output format from file extension.

```python
FORMAT_MAP = {
    ".obj": "obj",
    ".glb": "gltf",
    ".gltf": "gltf",
    ".stl": "stl",
    ".dae": "collada",
    ".skp": "skp",
    ".blend": "blend",
    ".ply": "ply",
}

class FormatDetector:
    @staticmethod
    def detect(format_name: str) -> str | None:
        return FORMAT_MAP.get(format_name.lower())

    @staticmethod
    def get_exporter(format_name: str) -> type | None:
        """Return the exporter class for a format."""
```

## Which exporter to use

When converting, the user specifies the target format:

```python
# In CadConverter.convert()
if output_format == "dxf":
    return DxfExporter().export(elements)
elif output_format == "ifc":
    return IfcExporter().export(elements)
elif output_format in ("obj", "glb", "gltf", "stl", "dae"):
    meshes = MeshExtruder().extrude_all(elements)
    exporter = FORMAT_EXPORTERS[output_format]()
    return exporter.export(meshes, output_path)
elif output_format == "skp":
    # Export COLLADA first, then convert
    collada_path = ColladaExporter().export(meshes, temp_path)
    return SkpExporter().export_from_collada(collada_path, output_path)
elif output_format == "blend":
    return BlendExporter().export(meshes, output_path)
```

## Limitations

- Extrusion uses fixed default heights — no storey detection from drawings
- No textures/materials — white/gray default materials only
- SketchUp export requires COLLADA intermediate (not native SKP writing)
- Blender export generates a Python script or uses headless Blender (requires Blender installed)
- No UV mapping — faces won't texture-map correctly without manual UV unwrap
- No boolean operations — intersecting walls create overlapping geometry
