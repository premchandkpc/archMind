# 11.2 — Geometry Building + DXF Export

**Status:** Planned

---

## What this chunk does

Takes `DetectedPrimitives` from Chunk 11.1 and builds structured CAD geometry:
- Merge lines into polylines
- Classify primitives into building elements (wall, column, beam, door, window, slab)
- Assign AIA-standard layers
- Export to DXF R12 format via `ezdxf`

Output: `.dxf` file openable in AutoCAD, LibreCAD, etc.

## Why split here

Geometry building is the core intelligence — turning raw lines into meaningful walls/columns/doors. DXF export is simpler but useful on its own before tackling IFC.

## Pipeline (chunk scope)

```
Input: DetectedPrimitives(lines, circles, symbols, texts)

  1. Merge touching collinear lines → polylines
  2. Detect walls → parallel line pairs
  3. Detect columns → hatched closed shapes
  4. Detect doors → gap-in-wall + arc symbol
  5. Detect windows → gap-in-wall + cross rectangle
  6. Detect beams → thick lines + nearby text
  7. Close shapes where confident (contour closure)
  8. Assign element type + layer
  9. DXF export (lines, polylines, circles, text on correct layers)
```

## Files to create

### civilmind/cad/geometry_builder.py

**Purpose:** Build structured `CadElement[]` from raw primitives.

```python
@dataclass
class CadElement:
    element_type: str  # "WALL" | "COLUMN" | "BEAM" | "SLAB" | "DOOR" | "WINDOW" | "GRID" | "UNKNOWN"
    geometry: list[Line | Polyline | Circle]
    layer: str
    properties: dict[str, str | float]
    bounds: tuple[float, float, float, float]  # xmin, ymin, xmax, ymax

class GeometryBuilder:
    """Build CadElements from detected primitives."""

    # Heuristic thresholds
    WALL_PARALLEL_DISTANCE_MIN = 100   # mm (assuming ~1px=1mm)
    WALL_PARALLEL_DISTANCE_MAX = 300
    WALL_MIN_LENGTH = 1000             # mm
    COLUMN_MIN_SIZE = 200              # mm
    COLUMN_MAX_SIZE = 600
    BEAM_LINE_WIDTH_MIN = 3            # px (thick lines)

    def build(
        self,
        primitives: DetectedPrimitives,
    ) -> list[CadElement]:
        """Run all detection heuristics and return classified elements."""
        elements = []
        elements += self._detect_walls(primitives.lines)
        elements += self._detect_columns(primitives.lines, primitives.symbols)
        elements += self._detect_doors(primitives.lines, primitives.symbols, primitives.texts)
        elements += self._detect_windows(primitives.lines, primitives.symbols)
        elements += self._detect_beams(primitives.lines, primitives.texts)
        elements += self._add_remaining_as_draft(primitives)
        return elements

    def _detect_walls(self, lines: list[Line]) -> list[CadElement]:
        """Find parallel line pairs → wall elements."""

    def _detect_columns(self, lines: list[Line], symbols: list[Symbol]) -> list[CadElement]:
        """Find closed hatched shapes → column elements."""

    def _detect_doors(self, lines: list[Line], symbols: list[Symbol], texts: list[TextBlock]) -> list[CadElement]:
        """Wall gap + arc + rectangle + label → door."""

    def _detect_windows(self, lines: list[Line], symbols: list[Symbol]) -> list[CadElement]:
        """Wall gap + rectangle with cross → window."""

    def _detect_beams(self, lines: list[Line], texts: list[TextBlock]) -> list[CadElement]:
        """Thick isolated lines with nearby annotation → beam."""
```

```python
def element_to_layer(el: CadElement) -> str:
    return {
        "WALL":   "A-WALL",
        "DOOR":   "A-DOOR",
        "WINDOW": "A-GLAZ",
        "COLUMN": "S-COLU",
        "BEAM":   "S-BEAM",
        "GRID":   "A-GRID",
    }.get(el.element_type, "A-DRAFT")

class DxfExporter:
    def export(self, elements: list[CadElement], output_path: str) -> str:
        """Export to DXF R12. Returns output_path."""
```

Entity mapping:
- `Line` → `msp.add_line()`
- `Polyline` → `msp.add_lwpolyline()`
- `Circle` → `msp.add_circle()`
- `TextBlock` → `msp.add_mtext()`

### civilmind/cad/polyline_merger.py

**Purpose:** Merge touching/nearby collinear line segments into polylines.

```python
class PolylineMerger:
    def merge(self, lines: list[Line], tolerance: float = 5.0) -> list[Polyline]:
        """Merge endpoint-connected collinear lines into polylines."""
```

Algorithm:
1. Build graph of line segments (nodes = endpoints, edges = lines)
2. For each connected component of collinear edges:
   - Sort by angle
   - Walk from one extreme endpoint to another
   - Collect ordered vertices
3. Return `list[Polyline]` (ordered vertex lists)

## Limitations

- Wall detection assumes orthogonal (horizontal/vertical) lines — rotated grids will be missed
- Door detection requires both gap + arc symbol — arc-only doors (single line arc) not detected
- No slab detection yet (requires hatch pattern understanding)
- DXF R12 only — no R2000+ entities (hatches, leaders, dim styles)
