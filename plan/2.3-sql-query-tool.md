## Security

- Read-only connection (no DDL, no DML)
- Parameterized queries only (no string interpolation)
- Query timeout: 5 seconds
- Result limit: 1000 rows
- Audit log every query

## Failure scenarios

| Scenario | Behavior |
|----------|----------|
| Write attempt | Rejected with "read-only" error |
| Slow query | Killed after 5s timeout |
| SQL injection | Parameterized queries prevent this |
