## What this chunk does

Extracts tables from PDFs and images.
Classifies tables as BOQ, spec, schedule, or cost breakdown.
Converts to structured JSON or pandas DataFrame.

## Why table extraction matters

- BOQs contain cost data (quantities, rates, amounts)
- Specifications contain material properties
- Schedules contain timelines
- Tables must NOT be chunked (they break)

## Table Types

| Type | Columns | Example |
|------|---------|---------|
| BOQ | Item, Qty, Unit, Rate, Amount | Concrete, Steel, Bricks |
| Spec | Material, Property, Value | M25, Grade, Strength |
| Schedule | Task, Duration, Start, End | Foundation, 15 days |
| Cost | Category, Subcategory, Cost | Materials, Labor |

## Extraction Strategies

### PDF Tables (pdfplumber)
```python
with pdfplumber.open(pdf_path) as pdf:
    for page in pdf.pages:
        for table in page.extract_tables():
            headers = table[0]
            rows = table[1:]
```

## BOQ Detection

```python
BOQ_KEYWORDS = [
    "bill of quantities", "boq", "schedule of rates",
    "item", "quantity", "unit", "rate", "amount",
]

def _is_boq(headers: list[str]) -> bool:
    header_text = " ".join(headers).lower()
    return sum(1 for kw in BOQ_KEYWORDS if kw in header_text) >= 3
```
