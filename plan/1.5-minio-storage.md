# 1.5 — MinIO Object Storage

**Status:** Done

## What this chunk does

Stores uploaded files (PDFs, drawings, images, Excel) in S3-compatible object storage.
Replaces the local filesystem storage from 1.2.
Every document upload saves to MinIO; every document read fetches from MinIO.

## Why object storage

| Storage type | Use case | Limitation |
|--------------|----------|------------|
| Local filesystem | Development only | No multi-user, no backup |
| S3 | Production | Cloud dependency, cost |
| MinIO | Self-hosted S3 | Needs Docker |

MinIO gives S3 compatibility without cloud dependency.

## Why MinIO

| Feature | MinIO | AWS S3 | Local FS |
|---------|-------|--------|----------|
| Self-hosted | Yes | No | Yes |
| S3-compatible | Yes | Native | No |
| Docker | Yes | N/A | N/A |
| Cost | Free | Paid ($0.023/GB) | Free |
| Multi-user | Yes | Yes | No |
| Presigned URLs | Yes | Yes | No |
| Versioning | Yes | Yes | No |

MinIO is the best choice for self-hosted object storage.

## Class API

```python
class MinIOStorage:
    def __init__(self, endpoint, access_key, secret_key, bucket, secure=False)
    
    async def ensure_bucket()                           # Create bucket if missing
    async def upload(file_bytes, key, content_type) -> str   # Upload file
    async def download(key) -> bytes                     # Download file
    async def delete(key)                                # Delete file
    async def list_files(prefix) -> list[FileInfo]       # List with prefix
    async def get_presigned_url(key, expires) -> str     # Temporary URL
    async def file_exists(key) -> bool                   # Check existence
    async def get_file_info(key) -> FileInfo | None      # Get metadata
    async def copy_file(source_key, dest_key)            # Copy within bucket
    async def get_file_bytes(key) -> tuple[bytes, str]   # Download + content type
```

## Storage Path Convention

```
projects/{project_id}/docs/{document_id}.pdf          # Uploaded documents
projects/{project_id}/docs/{document_id}_v2.pdf       # Re-uploaded (versioned)
projects/{project_id}/reports/{report_id}.pdf          # Generated reports
projects/{project_id}/temp/{uuid}.png                  # Temporary processing files
projects/{project_id}/drawings/{drawing_id}.png        # Extracted drawings
```

**Why this structure:**
- Organized by project (easy to delete all project files)
- Versioned by document_id (re-upload creates new version)
- Separate directories for different file types

## FileInfo Model

```python
@dataclass
class FileInfo:
    key: str              # e.g., "projects/p1/docs/d1.pdf"
    size: int             # bytes
    last_modified: datetime
    etag: str             # content hash (MD5)
    content_type: str     # e.g., "application/pdf"
```

## Method Details

### ensure_bucket
- Checks if bucket exists
- Creates if not
- Idempotent (safe to call multiple times)

### upload
- Accepts raw bytes, key, and content_type
- Creates MinIO PutObject with content type
- Returns the key

### download
- Accepts key
- Returns raw bytes
- Raises FileNotFoundError if key doesn't exist

### list_files
- Accepts prefix (e.g., "projects/p1/docs/")
- Returns list of FileInfo objects
- Sorted by key

### get_presigned_url
- Accepts key and expiry (seconds)
- Returns temporary download URL (no auth needed)
- Useful for sharing files with external users

### file_exists
- Accepts key
- Returns True/False
- Uses stat_object (efficient)

### get_file_info
- Accepts key
- Returns FileInfo or None
- Uses stat_object

## Usage Flow

```python
storage = MinIOStorage(endpoint="localhost:9000", ...)

# Ensure bucket exists
await storage.ensure_bucket()

# Upload file
with open("blueprint.pdf", "rb") as f:
    data = f.read()

key = await storage.upload(
    file_bytes=data,
    key="projects/proj-1/docs/doc-abc.pdf",
    content_type="application/pdf",
)

# Download file
data = await storage.download("projects/proj-1/docs/doc-abc.pdf")

# Generate temporary URL (for sharing)
url = await storage.get_presigned_url("projects/proj-1/docs/doc-abc.pdf", expires=3600)
# Returns: http://localhost:9000/civilmind-docs/projects/proj-1/docs/doc-abc.pdf?signature=...

# List project files
files = await storage.list_files("projects/proj-1/docs/")
for f in files:
    print(f"{f.key} ({f.size} bytes)")
```
