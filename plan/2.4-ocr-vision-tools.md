## What this chunk does

PaddleOCR for text extraction, Vision LLM for image analysis.
Both as tools with consistent ToolResult interface.

## Files to create

### civilmind/tools/ocr.py

```python
class OCRTool(BaseTool):
    name = "ocr"
    description = "Extract text from images and scanned documents"
    category = "vision"
    
    async def execute(self, image_path: str) -> ToolResult:
        """PaddleOCR text extraction."""
```

### civilmind/tools/vision_llm.py

```python
class VisionLLMTool(BaseTool):
    name = "vision_llm"
    description = "Analyze images using vision model"
    category = "vision"
    
    async def execute(self, image_path: str, prompt: str) -> ToolResult:
        """OpenCode Zen vision model analysis."""
```

## Failure scenarios

| Scenario | Behavior |
|----------|----------|
| OCR worker down | ToolResult(success=False, error="OCR unavailable") |
| Image too large | Reject (>20MB) |
| Vision API error | Retry once, then fail |
