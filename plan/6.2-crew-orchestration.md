## What this chunk does

Assembles the CrewAI Crew with agents and tasks.

Shared memory across crew for context continuity.

## Execution Flow

```
Crew.kickoff(question="Estimate concrete")
  │

  │
  ├─→ Compliance Agent

  │

  │     └─ Calculates quantities
  │
  ├─→ Scheduler Agent

  │
  ├─→ Risk Analyzer

  │
  ├─→ Reviewer Agent

  │

```
