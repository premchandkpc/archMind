# 6.1 — Agent Definitions

**Status:** Done

## What this chunk does

Defines 9 specialized AI agents with roles, goals, backstories, and tools.
Each agent is an expert in one domain and collaborates through CrewAI.

## Why CrewAI agents

- Role-based design (clear responsibilities)
- Tool integration (agents can call external functions)
- Delegation (planner can ask other agents for help)
- Memory (agents remember past interactions)
- Planning (agents can break down complex tasks)

## Files to create

### civilmind/agents/roles.py

**Purpose:** AgentFactory with 9 create_*() methods.

**Purpose:** 8 tool implementations.

### CalculatorTool
```python
class CalculatorTool(BaseTool):
    name: str = "calculator"
    description: str = "Perform mathematical calculations"
    
    def _run(self, expression: str) -> str:
        # Safe evaluation of math expressions
        result = eval(expression, {"__builtins__": {}}, {})
        return str(result)
```

## Agent Backstories

Each agent has a detailed backstory that grounds their expertise:

**Planner:** "Senior construction project manager with 20 years experience planning residential and commercial projects."

**Retriever:** "Expert at finding information in construction documents. Knows where to look for structural specs, building codes, material properties."

**Drawing Analyzer:** "Civil engineer specializing in reading architectural and structural drawings. Can identify walls, columns, beams, doors, windows."

**Compliance:** "Government building inspector with deep knowledge of Indian building codes. Knows every clause of NBC 2016, IS456, IS875."

**Estimator:** "Certified quantity surveyor with expertise in Indian construction costs. Can estimate concrete, steel, bricks, labor."

**Scheduler:** "Planning engineer who has created schedules for hundreds of construction projects. Understands dependencies, critical paths."

**Risk Analyst:** "Risk management professional specializing in construction. Knows common pitfalls, cost overrun patterns."

**Reviewer:** "Senior engineer who reviews all deliverables. Catches calculation errors, missing information, hallucinations."

**Report Writer:** "Technical writer who creates construction reports for architects, engineers, and government departments."
