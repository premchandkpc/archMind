# 11.3 — IFC Export + Converter Orchestration

**Status:** Planned

---

## What this chunk does

- Export `CadElement[]` to IFC 2x3 / IFC4 BIM format
- Orchestrate the full conversion pipeline: rasterize → detect → build geometry → export
- Status tracking (progress callbacks)

## Dependency

Requires Chunks 11.1 (primitives) and 11.2 (geometry + DXF) to be done. This chunk wires them together and adds IFC.

## Pipeline (chunk scope)

```
Input: file_path (PDF/PNG/JPG)

  1. Rasterize       → np.ndarray          [0-20%]
  2. Detect lines    → Line[] + Circle[]    [20-40%]
  3. Symbol + text   → Symbol[] + Text[]    [40-55%]
  4. Build geometry  → CadElement[]         [55-75%]
  5. Export to IFC   → .ifc file            [75-95%]
  6. Cleanup         → output path          [95-100%]
```

## Files to create

### civilmind/cad/ifc_exporter.py

**Purpose:** Export `CadElement[]` to IFC2x3 format.

```python
class IfcExporter:
    """Export CadElements to IFC BIM format."""

    # Default height for extruded elements (mm)
    DEFAULT_WALL_HEIGHT = 3000
    DEFAULT_COLUMN_HEIGHT = 3000
    DEFAULT_BEAM_HEIGHT = 450
    DEFAULT_SLAB_THICKNESS = 200

    def export(self, elements: list[CadElement], output_path: str) -> str:
        """Export to IFC2x3. Returns output_path."""
```

Entity mapping:
| CadElement type | IfcEntity | Extrusion |
|---|---|---|
| WALL | IfcWall | Extrude 2D footprint by DEFAULT_WALL_HEIGHT |
| COLUMN | IfcColumn | Extrude 2D footprint by DEFAULT_COLUMN_HEIGHT |
| BEAM | IfcBeam | Extrude thick line path by DEFAULT_BEAM_HEIGHT |
| DOOR | IfcDoor | Create opening in wall, add door panel |
| WINDOW | IfcWindow | Create opening in wall, add window frame |
| SLAB | IfcSlab | Extrude closed contour by DEFAULT_SLAB_THICKNESS |

IFC structure:
```
IfcProject
├── IfcSite
│   └── IfcBuilding
│       ├── IfcWall            (one per wall element)
│       ├── IfcColumn          (one per column element)
│       ├── IfcBeam            (one per beam element)
│       ├── IfcDoor            (one per door element)
│       ├── IfcWindow          (one per window element)
│       └── IfcSlab            (one per slab element)
```

Each element gets:
- `IfcLocalPlacement` (2D position + extrusion direction)
- `IfcExtrudedAreaSolid` (footprint + height)
- `IfcPropertySet` with extracted text annotations (e.g., "M25", "230x450")

Implementation approach:
```python
import ifcopenshell
import ifcopenshell.api as ifcapi

model = ifcopenshell.file()
project = ifcapi.run("root.create_entity", model, ifc_class="IfcProject",
    name="ConvertedDrawing")

# Geometric representation context
context = ifcapi.run("context.add_context", model,
    context_type="Model", context_identifier="Body")

# For each 2D footprint, we create an IfcArbitraryClosedProfileDef
# and extrude it vertically using IfcExtrudedAreaSolid

# Column example:
footprint = model.create_entity("IfcPolyline", [
    model.create_entity("IfcCartesianPoint", coords) for coords in footprint_points
])
profile = model.create_entity("IfcArbitraryClosedProfileDef",
    "AREA", None, footprint)
extrusion = model.create_entity("IfcExtrudedAreaSolid",
    profile, placement, direction, height)

# Set placement:
placement = model.create_entity("IfcLocalPlacement",
    relative_placement=model.create_entity("IfcAxis2Placement3D",
        model.create_entity("IfcCartesianPoint", (x, y, z))))
```

### civilmind/cad/converter.py

**Purpose:** Orchestrate the full conversion pipeline.

```python
class CadConverter:
    """Full pipeline: file → primitives → geometry → DXF/IFC."""

    def __init__(
        self,
        detector: LineDetector | None = None,
        symbol_recognizer: SymbolRecognizer | None = None,
        geometry_builder: GeometryBuilder | None = None,
        dxf_exporter: DxfExporter | None = None,
        ifc_exporter: IfcExporter | None = None,
    ):
        # ... DI for testability

    async def convert(
        self,
        file_path: str,
        output_format: str = "dxf",      # "dxf" | "ifc"
        scale_mm_per_pixel: float = 1.0,  # auto-detect if 0
        on_progress: Callable[[float, str], None] | None = None,
    ) -> CadConversionResult:
        """Convert a drawing file to CAD/BIM format.

        Returns CadConversionResult with output path, element count, etc.
        """

    async def _rasterize(self, file_path: str) -> np.ndarray:
        """Convert PDF/PNG/JPG to grayscale numpy array."""

    def _export(self, elements: list[CadElement], fmt: str) -> str:
        """Dispatch to DxfExporter or IfcExporter."""

    async def _auto_detect_scale(self, image: np.ndarray, texts: list[TextBlock]) -> float:
        """Try to extract scale from drawing title block or dimension text."""


@dataclass
class CadConversionResult:
    success: bool
    output_path: str | None = None
    element_count: int = 0
    wall_count: int = 0
    column_count: int = 0
    door_count: int = 0
    window_count: int = 0
    beam_count: int = 0
    error: str | None = None
```

## Limitations

- All elements use default height — no storey/floor detection
- No material mapping — text labels like "M25" stored as IfcPropertySet, not IfcMaterial
- No openings — doors/windows are separate IfcDoor/IfcWindow, not IfcOpeningElement in walls
- IFC2x3 only — IFC4 support requires separate entity generation
