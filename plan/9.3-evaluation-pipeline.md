## What this chunk does

Automated evaluation against benchmark datasets.
Run after each code change to catch regressions.

## Files to create

### civilmind/evaluation/benchmarks.py

```python
class BenchmarkRunner:
    async def run(self, dataset: list[BenchmarkCase]) -> BenchmarkReport:
        """Run evaluation on benchmark dataset."""
    
    async def run_single(self, case: BenchmarkCase) -> BenchmarkResult:
        """Evaluate a single query."""

@dataclass
class BenchmarkCase:
    query: str
    expected_answer: str
    expected_chunks: list[str]  # chunk IDs
    domain: str

@dataclass
class BenchmarkReport:
    total_cases: int
    retrieval_recall: float
    retrieval_precision: float
    faithfulness_score: float
    answer_relevance: float
    avg_latency_ms: float
    total_cost_usd: float
    failed_cases: list[BenchmarkResult]
```

## Benchmark Dataset Format

```json
[
  {
    "query": "What is the minimum concrete grade for residential buildings?",
    "expected_answer": "M20 is the minimum grade per IS 456",
    "expected_chunks": ["chunk_is456_5_3", "chunk_is456_table_5"],
    "domain": "civil_engineering"
  }
]
```

## CI Integration

```yaml
# After each PR
- name: Run evaluation
  run: python -m civilmind.evaluation.benchmarks --dataset benchmark.json

- name: Check regression
  run: |
    if [ $RECALL_AT_5 < 0.8 ]; then exit 1; fi
    if [ $FAITHFULNESS < 0.85 ]; then exit 1; fi
```
