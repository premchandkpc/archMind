## What this chunk does

Parses uploaded documents (PDF, DOCX, Excel, images) into structured elements.
This is the first step of the document pipeline: raw files → structured elements → chunks → embeddings → vectors.

## Why structured parsing matters

A PDF is not just text. It contains:
- **Text paragraphs** (narrative content) → need semantic chunking
- **Tables** (BOQ, specifications) → must NOT be split (they break)
- **Images** (floor plans, diagrams) → need vision model analysis
- **Titles** (section headers) → become metadata for chunks

Each type needs different handling downstream.

## Files to create

### civilmind/pipeline/parser.py

**Purpose:** DocumentParser class that converts raw files into ParsedElement objects.

## Supported Formats

| Format | Parser | Output elements | Dependencies |
|--------|--------|-----------------|--------------|
| `.pdf` | Unstructured partition_pdf | Text, tables, images | pdfminer, pdf2image |
| `.docx` | Unstructured partition_docx | Text, tables | python-docx |
| `.xlsx` | pandas.read_excel | Tables (one per sheet) | openpyxl |
| `.png/.jpg` | PaddleOCR | OCR text + image reference | paddleocr |

## ElementType Enum

```python
class ElementType(Enum):
    TITLE = "title"         # Section headers (e.g., "Foundation Details")
    NARRATIVE = "narrative" # Body text paragraphs
    TABLE = "table"         # Tabular data (BOQ, specs)
    IMAGE = "image"         # Images and diagrams
    LIST = "list"           # Bullet/numbered lists
    PAGE_BREAK = "page_break"  # Page separators (skipped)
    UNKNOWN = "unknown"     # Unclassified elements
```

## ParsedElement Model

```python
@dataclass
class ParsedElement:
    type: ElementType              # What kind of element
    content: str                   # Text content or description
    metadata: dict                 # page_number, char_count, word_count, etc.
    table_data: list[list[str]] | None  # For tables: [[header1, header2], [val1, val2]]
    image_path: str | None        # For images: path to saved image file
    page_number: int | None       # Page number in original document
```

## PDF Parsing Flow

```
PDF file
  → Unstructured.partition_pdf(strategy="hi_res")
    → split_into_elements
    → For each element:
        NarrativeText → ElementType.NARRATIVE
        Title → ElementType.TITLE
        Table → ElementType.TABLE
        Image → ElementType.IMAGE
        ListItem → ElementType.LIST
        PageBreak → skip
    → Extract metadata from element.metadata:
        page_number, element_index
    → For images: save to /tmp/civilmind_images/, store path
    → For tables: convert to list[list[str]]
    → Return list[ParsedElement]
```

## Table Detection

Tables are detected by Unstructured's built-in table extraction.
If Unstructured fails to detect a table, we use heuristics:
- Element type is Table
- Contains `|` or `\t` separators
- Has consistent column count across rows

## Image Extraction

```python
# Images are extracted from PDF and saved
partition_pdf(
    filename=file_path,
    extract_images_in_pdf=True,           # Extract embedded images
    image_output_dir_path="/tmp/civilmind_images",  # Save here
    strategy="hi_res",                     # High-quality extraction
)
```

## OCR for Scanned Documents

If a PDF page has no text layer (scanned document):
1. Unstructured returns empty text for that page
2. We fall back to PaddleOCR
3. OCR extracts text from the image
4. Confidence threshold: 0.5 (ignore low-confidence text)

## Edge Cases

| Case | Handling |
|------|----------|
| Scanned PDF (no text) | Falls back to PaddleOCR |
| Password-protected PDF | Raises ValueError |
| Corrupted file | Catches exception, logs error, returns empty list |
| Empty pages | Skipped (no element created) |
| Very large images | PaddleOCR handles resizing internally |
| Tables embedded in text | Detected by Unstructured's table extractor |
| Mixed languages | PaddleOCR supports multiple languages |

## Usage

```python
parser = DocumentParser()

# Parse PDF
elements = await parser.parse("samples/building_plan.pdf")
for e in elements:
    print(f"{e.type.value}: {e.content[:80]}...")

# Parse DOCX
elements = await parser.parse("samples/specifications.docx")

# Parse Excel
elements = await parser.parse("samples/boq.xlsx")
# Returns: [ParsedElement(type=TABLE, content="Sheet: BOQ", table_data=[...])]

# Parse Image
elements = await parser.parse("samples/floor_plan.png")
# Returns: [ParsedElement(type=IMAGE, ...), ParsedElement(type=NARRATIVE, content="OCR text...")]
```
