# 5.1 — OCR + Floor Plan Analysis

**Status:** Pending

## What this chunk does

PaddleOCR extracts text from scanned documents.
Vision LLM (via OpenCode Zen) analyzes floor plans to detect walls, columns, beams, doors, windows, rooms, and dimensions.

## Why vision matters

- Floor plans contain structural information not in text
- OCR extracts text from scanned PDFs
- Vision models understand spatial relationships
- Enables "find conflicts between plumbing and structural beams"

## Files to create

### civilmind/vision/ocr.py

**Purpose:** OCREngine wrapping PaddleOCR.

### civilmind/vision/floorplan.py

**Purpose:** FloorPlanAnalyzer using OpenCode Zen vision models.

## OCR Engine

```python
class OCREngine:
    def __init__(self, lang="en")
    async def extract_text(image_path) -> list[OCRResult]
    async def extract_all_text(image_path) -> str
```

### OCRResult Model

```python
@dataclass
class OCRResult:
    text: str              # Extracted text
    confidence: float      # 0-1 confidence score
    bbox: list[list[int]]  # Bounding box coordinates
    page: int | None       # Page number (for PDFs)
```

## Vision Models Comparison

| Model | Use Case | Cost | Quality |
|-------|----------|------|---------|
| OpenCode Zen (GPT-5, Claude) | General analysis | Pay-per-token | Excellent |
| Florence-2 | Object detection | Free | Good |
| Grounding DINO | Object detection | Free | Good |
| PaddleOCR | Text extraction | Free | Good |

**Recommendation:** OpenCode Zen for high-level analysis, PaddleOCR for text.
