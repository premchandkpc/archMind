## What this chunk does

Defines the abstract tool interface and DI container.
Agents never query databases directly — they call tools.
Tools are swappable, testable, and independently deployable.

## Why a Tool Layer

| Without Tool Layer | With Tool Layer |
|-------------------|-----------------|
| Agent directly calls Qdrant client | Agent calls VectorSearchTool.execute() |
| Testing requires live Qdrant | Testing uses mock tool |
| Swapping Qdrant → Pinecone changes agent code | Swapping changes tool implementation only |
| No latency tracking per operation | ToolResult includes latency_ms |
| No token counting | ToolResult includes tokens_used |

## Files to create

### civilmind/tools/base.py

**Purpose:** Abstract base class and result type.

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass, field
from typing import Any

@dataclass
class ToolResult:
    success: bool
    data: Any = None
    error: str | None = None
    latency_ms: float = 0.0
    tokens_used: int | None = None
    metadata: dict = field(default_factory=dict)

class BaseTool(ABC):
    name: str
    description: str
    category: str  # retrieval, calculation, vision, knowledge
    
    @abstractmethod
    async def execute(self, **kwargs) -> ToolResult: ...
    
    async def __call__(self, **kwargs) -> ToolResult:
        return await self.execute(**kwargs)
```

### civilmind/tools/registry.py

**Purpose:** Tool registry with DI.

```python
class ToolRegistry:
    def __init__(self):
        self._tools: dict[str, BaseTool] = {}
    
    def register(self, tool: BaseTool) -> None:
        self._tools[tool.name] = tool
    
    def get(self, name: str) -> BaseTool:
        return self._tools[name]
    
    def list_tools(self) -> list[dict]:
        return [{"name": t.name, "description": t.description, "category": t.category} 
                for t in self._tools.values()]

# Global registry
registry = ToolRegistry()
```

### civilmind/tools/dependencies.py

**Purpose:** FastAPI dependency for tool injection.

```python
async def get_tool_registry() -> ToolRegistry:
    return registry
```

## Failure scenarios

| Scenario | Behavior |
|----------|----------|
| Unknown tool name | KeyError with helpful message |
| Tool execution timeout | ToolResult(success=False, error="Timeout") |
| Tool raises exception | Caught, wrapped in ToolResult |
