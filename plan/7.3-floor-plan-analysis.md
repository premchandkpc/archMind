```json

  "walls": [{"type": "load_bearing", "thickness_mm": 230}],
  "doors": [{"id": "D1", "width_mm": 900}],
```

## Failure scenarios

| Scenario | Behavior |
|----------|----------|
| Image too small | Upscale before analysis |
| Vision API error | Retry once, then return partial results |
| Ambiguous elements | Mark confidence < 0.5 |
