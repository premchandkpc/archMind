## What this chunk does

Structured logging, Prometheus metrics, OpenTelemetry tracing.
Without observability, you can't debug production issues.

## Three Pillars

| Pillar | Tool | Purpose |
|--------|------|---------|
| Logs | structlog | Structured, searchable, contextual |
| Metrics | prometheus_client | Counters, histograms, gauges |
| Traces | opentelemetry | Request flow across services |

### civilmind/infrastructure/observability/logging.py

```python
import structlog

def configure_logging():
    structlog.configure(
        processors=[
            structlog.contextvars.merge_contextvars,
            structlog.processors.add_log_level,
            structlog.processors.TimeStamper(fmt="iso"),
            structlog.dev.ConsoleRenderer() if DEBUG else structlog.processors.JSONRenderer(),
        ],
    )
```

### civilmind/infrastructure/observability/metrics.py

```python
from prometheus_client import Counter, Histogram, Gauge

# Tool metrics
tool_calls_total = Counter("tool_calls_total", "Tool calls", ["tool_name", "status"])
tool_latency_seconds = Histogram("tool_latency_seconds", "Tool latency", ["tool_name"])

# Pipeline metrics
pipeline_events_total = Counter("pipeline_events_total", "Pipeline events", ["event_type", "status"])
pipeline_queue_depth = Gauge("pipeline_queue_depth", "Queue depth", ["stream"])

# LLM metrics
llm_calls_total = Counter("llm_calls_total", "LLM calls", ["model", "status"])
llm_tokens_total = Counter("llm_tokens_total", "LLM tokens", ["model", "type"])
llm_cost_usd = Counter("llm_cost_usd", "LLM cost", ["model"])

# Retrieval metrics
retrieval_latency_seconds = Histogram("retrieval_latency_seconds", "Retrieval latency")
retrieval_results_count = Histogram("retrieval_results_count", "Retrieval result count")

# Query metrics
query_duration_seconds = Histogram("query_duration_seconds", "Query duration")
query_total = Counter("query_total", "Queries", ["status"])
```

### civilmind/infrastructure/observability/tracing.py

```python
from opentelemetry import trace
from opentelemetry.sdk.trace import TracerProvider
from opentelemetry.sdk.trace.export import BatchSpanExporter
from opentelemetry.exporter.otlp.proto.grpc.trace_exporter import OTLPSpanExporter

def configure_tracing():
    provider = TracerProvider()
    exporter = OTLPSpanExporter()
    provider.add_span_processor(BatchSpanExporter(exporter))
    trace.set_tracer_provider(provider)
```

## Alerting Rules

| Alert | Condition | Severity |
|-------|-----------|----------|
| High error rate | error_rate > 5% for 5min | critical |
| High latency | p99_latency > 30s for 5min | warning |
| Queue depth | queue_depth > 100 for 10min | warning |
| LLM cost spike | cost_rate > $10/hour | warning |
| Worker down | workers_active < 1 for 2min | critical |

## Security

- Never log API keys or secrets
- Structured logs shipped to centralized logging (ELK/Loki)
- Metrics endpoint on separate port (9090)
- Traces sampled at 10% in prod (adjustable)
