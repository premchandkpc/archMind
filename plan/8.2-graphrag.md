## What this chunk does

Combines vector search with graph traversal.
Finds relevant chunks via vector search, then expands via graph relationships.
Builds rich context for LLM answers.

## Why GraphRAG

- Vector search finds text mentions
- Graph traversal finds relationships
- Combined context is richer than either alone
- Enables multi-hop questions

### civilmind/graph/graphrag.py

**Purpose:** GraphRAG retrieval + answer generation.

## GraphRAG Pipeline

```
Query "What vendor supplies concrete for Bedroom 1?"
  → Vector search: find chunks mentioning "vendor", "concrete", "Bedroom"
  → Extract entity IDs from chunks
  → Graph traversal (2 hops):
      Material {type: "concrete"} —SUPPLIED_BY→ Vendor
      Room {name: "Bedroom"} —HAS_WALL→ Wall —USES_MATERIAL→ Material
  → Combine context
  → LLM answer with evidence
```
